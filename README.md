# 📊 E-Commerce Dashboard with Google Looker Studio

## Project Overview

This project was completed as the Final Project of the MySkill Data Analyst Bootcamp.

The objective of this project is to build an interactive dashboard using Google Looker Studio to monitor business performance, analyze customer purchasing behavior, evaluate product performance, and support data-driven business decisions.

The dashboard provides interactive visualizations that help stakeholders monitor key business metrics and gain actionable insights from e-commerce transaction data.

## 📋 Business Questions

### Question 1

Create an interactive dashboard that displays the following business KPIs:

- Total Sales Before Discount
- Total Sales After Discount
- Net Profit
- Quantity Sold
- Total Customers
- Average Order Value (AOV)

---

### Question 2

Analyze customer transactions by applying filters such as:

- Date Range
- Product Category
- Transaction Status
- Payment Method

The analysis found that no valid transactions were recorded for the **Mobile & Tablets** category using **JazzWallet** as the payment method.

## 💡 Key Insights

- Cash on Delivery (COD) is the most frequently used payment method.
- JazzWallet is the second-largest digital payment method.
- Repeat purchases contribute significantly to total revenue.
- No valid Mobile & Tablets transactions were made using JazzWallet within the selected period.

---

## 📌 Business Recommendations

- Increase cashback promotions for JazzWallet users.
- Develop loyalty programs for repeat customers.
- Encourage customers to switch from COD to digital payment methods.
- Launch exclusive campaigns targeting high-value customers.

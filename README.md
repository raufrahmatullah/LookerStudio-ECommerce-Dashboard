# 📊 E-Commerce Dashboard with Google Looker Studio

## Project Overview

This project was completed as the Final Project of the MySkill Data Analyst Bootcamp.

The objective of this project is to build an interactive dashboard using Google Looker Studio that helps stakeholders monitor business performance, analyze customer purchasing behavior, and support data-driven decision making.

The dashboard focuses on sales performance, customer metrics, product performance, and payment method analysis.

## Business Questions

### Question 1

Build an interactive dashboard that displays:

- Sales Value
- Net Profit
- Quantity Sold
- Customer Count
- Average Order Value (AOV)
- Payment Method Distribution

---

### Question 2

Analyze transactions using filters such as:

- Date
- Product Category
- Transaction Status
- Payment Method

The analysis found that there were no valid Mobile & Tablets transactions using JazzWallet as the payment method.

## Business Insights

Some key findings include:

- Cash on Delivery (COD) remains the dominant payment method.
- JazzWallet is the second-largest digital payment method.
- Repeat purchases contribute significantly to total revenue.
- Mobile & Tablets transactions using JazzWallet were not found in the selected analysis.

---

## Recommendations

- Increase cashback promotions for JazzWallet users.
- Create loyalty programs for repeat buyers.
- Encourage customers to switch from COD to digital payments.
- Launch exclusive campaigns for high-value customers.
